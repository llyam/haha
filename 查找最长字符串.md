## 1.查找最长字符串

#### （1）题目说明

遍历该文件夹下所有文件夹，在所有文件夹的目录中查找最长的字符串的一行，输出长度，该字符串及路径。

#### （2）代码说明

```c
#!/bin/bash
sum=''
max=''
long=''
s=0
find() {
    for i in `ls`; do  //遍历文件
        if [[ -d $i ]]; then //是文件则进入
            cd      
            find   //递归
            cd ..
            continue
        fi
        if [[ -f $i ]]; then  //是目录
            for j in `cat $i`; do //遍历行
                s=${#j}
                if [[ s > max ]]; then
                    max=$s
                    sum=$j
                    long=`pwd`
                fi
            done
        fi
    done
}
cd $1
find 
echo $max  //长度
echo $sum  //字符串
echo $long //路径
```
