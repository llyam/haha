## scapry

远端的进行下载之后分析

1. 下载器（download）
2. 分析器（parse）
3. 存储器（store， save， data， item（一项， 条目））

#### 新建爬虫项目

首先， terminal  先看strapy

conda active scrapy

scrapy --help

```
（1）
scrapy startproject + 爬虫项目的名字
（2）
新建爬虫  scrapy genspider + 名字 + 网址
进入新建的文件内 查看
（3）
scrapy crawl + 爬虫名字  进行爬取
××进行user-agent配置××
 网站+/rebots.txt 查看可以爬到的 需要遵守的
××配置完成××
```
